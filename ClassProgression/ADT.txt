ADT Прогрессии

Данные
    • вид прогресии: ГП или АП  //перечесляемый тип num
    • шаг прогрессии(step): веществ;
    • первый элемент(FirstE): веществ;  


Операции
    • Конструктор
		Начальные значения: 0, 0, АП
		Процесс: Шаг = 0, Первый элемент АП = 0, вид прогрессии=АП


    • Операция Найти значение n-го элемента
		Вход: Номер находимого элемента
		Предусловия: Номер является целым положительным числом
		Процесс: Находимый_элемент= FirstE *step-1 или Находимый_элемент =   FirstE * step * (n-1);
		Выход: Значение n-го элемента 
		Постусловия:  —

    • Операция Найти сумму n первых элементов 
		Вход: Кол-во элементов
		Предусловия: Кол-во элементов является целым положительным числом
		Процесс: Сумма = ( FirstE*(1-stepn))/(1-step) или Сумма=(2* FirstE +step*(Кол-во_элементов-1))/2
		Выход: Сумма n первых элементов 
		Постусловия: —


    • Операция Задать первый элемент прогрессии
		Вход: Новое_значение_первого_элемента 
		Предусловия: —
		Процесс:  Задание первого элемента
		Выход: —
		Постусловия: —

    • Операция Задать шаг прогрессии
		Вход: Новый_шаг_прогрессии
		Предусловия: —
		Процесс:  Задание разности прогрессии
		Выход: —
		Постусловия: —

    • Операция Задать тип прогрессии
		Вход: Новый_вид_прогрессии
		Предусловия: —
		Процесс:  Задание вида прогрессии
		Выход: —
		Постусловия: —

    • Операция Чтение значения первого элемента
		Вход: —
		Предусловия: —
		Процесс:  прочитать значение первого элемента
		Выход: Первый_элемент
		Постусловия: —

    • Операция Чтение значение шага прогрессии
		Вход: —
		Предусловия: —
		Процесс:  прочитать значение шага прогрессии
		Выход: Шаг_прогрессии
		Постусловия: —
          
    • Операция Чтение вида прогрессии
		Вход: —
		Предусловия: —
		Процесс:  прочитать вид прогрессии
		Выход: Вид_прогрессии
		Постусловия: —




    
Конец ADT Прогрессии
 













Функции:

     • Операция Определить является последовательность АП
	Вход: Последовательность вещественных чисел
	Предусловия: Кол-во_элементов>1
	Процесс: проверка соблюдения условия Элемент_n-Элемент_n-1=Разность_АП (найдена как Разность_АП=Второй_элемент-Первый_элемент) для всех элементов 
Выход: истина или ложь
	Постусловия: —
      • Операция Определить является последовательность ГП
	Вход: Последовательность вещественных чисел
	Предусловия: Кол-во_элементов>1, нет элементов равных нулю
	Процесс: проверка соблюдения условия Элемент_n/Элемент_n-1=Знаменатель_ГП (найдена как Знаменатель_ГП=Второй_элемент/Первый_элемент) для всех элементов 
Выход: истина или ложь
	Постусловия: —
       • Операция Нахождение разности АП
	Вход: Последовательность вещественных чисел
	Предусловия: Кол-во_элементов>1, это последовательность АП
	Процесс: Разность_АП=Второй_элемент-Первый_элемент
Выход: Разность_АП
	Постусловия: —
        • Операция Нахождение знаменателя ГП
	Вход: Последовательность вещественных чисел
	Предусловия: Кол-во_элементов>1, это последовательность ГП
	Процесс: Знаменатель_ГП=Второй_элемент/Первый_элемент
Выход: Знаменатель_ГП
	Постусловия: —
